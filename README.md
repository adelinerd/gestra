# GESTRA
This repository has some templates to be used in **GESTRA** related repos

## Issue Templates

These templates help to make sure, all the issues created in gitlab have the same topics in the description


## Merge Templates

These templates help to document all changes, including the information whether AI has been used


## Policies

- `LLM_USAGE_POLICY.md` can be used in the repos, that have been classified as "YELLOW"  with some rules for LLM usage.
